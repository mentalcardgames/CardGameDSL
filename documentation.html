How to use the library

players!(cgm: &mut CardGameModel, playername1: String, playername2: String, ...) => Takes a reference of a CardGameModel (cgm) and a player-names and SETs and INITIALIZEs the PLAYERs in cgm.

team!(cgm: &mut CardGameModel, teamname: String, (player1: Rc<RefCell<Player>>, player2: Rc<RefCell<Player>>, ...)) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs the a TEAM in cgm.

location_on!(locationname: String, players: Vec<Rc<RefCell<Player>>>) => Takes a location-name and a Vector of "Player-References" and SETs and INITIALIZEs the Locations on the Vector of Players. location_on!(locationname: String, team: &Team) => Takes a location-name and a "Team-Reference" and SETs and INITIALIZEs the Locations on the TEAM. location_on!(locationname: String, table: &Table) => Takes a location-name and a "Table-References" and SETs and INITIALIZEs the Locations on the TABLE.

card_on!(cgm: &mut CardGameModel, location: String, ...) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs CARDs on the specified LOCATION in cgm.

precedence!(cgm: &mut CardGameModel, (Key: String, (v1: String, v2: String, ...))) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs the PRECEDENCE for the given KEY in cgm.

pointmap!(cgm: &mut CardGameModel, nested: (keyname, ( valname1 => [i32], valname2 => [i32], ... ) ) ) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs the POINTMAP for the given KEY in cgm. pointmap!(cgm: &mut CardGameModel, list: ( (keyname1, valname1) => [i32], (keyname2, valname2) => [i32], (keyname3, valname3) => [i32], ... ) ) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs the POINTMAP for the given (KEY, VALUE)-Pairs in cgm.

turnorder!(cgm: &mut CardGameModel, players: Vec<Rc<RefCell<Player>>>) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs the TURNORDER for the given "Player-References" (in the order of the "Player-References") in cgm. turnorder!(cgm: &mut CardGameModel, players: Vec<Rc<RefCell<Player>>>, random) => Takes a reference of a CardGameModel (cgm) and SETs and INITIALIZEs a "random" TURNORDER for the given "Player-References" in cgm.
